  • The border-collapse cədvəldə xanaların bir-birinə birləşik və ya ayrı olmasını təyin edir
  • The border-spacing xanalar arasındakı boşluqları vermək üçün istifadə olunur.
  • The caption-side cədvələ verilən başlığın harda olmasını təyin edir.Default olaraq top olur yeri.
  • The empty-cells boş olan xanaları gizlətmək üçün istifadə olunur.

Border xassələri elementi təmsil edən qutunun sərhədinin necə görünəcəyini müəyyən etməyə imkan verir. Dəyişdirə biləcəyiniz border-da üç xüsusiyyət var:

  • The border-color kənarlığın rəngini dəyişmək üçün istifadə olunur.
  • The border-style kənarlığın qırıq-qırıq , düz xətt və s formasında olmasını təyin edir.
  • The border-width kənarlığın genişliyini təyin edir.

Border-color

The border-color Property

This property specifies the color of the four borders:

A solid red border

A solid green border

A dotted blue border

. Xüsusiyyətlərdən istifadə edərək elementin kənarlığın alt, sol, yuxarı və sağ tərəflərinin rəngini fərdi olaraq dəyişə bilərsiniz

Border-style

Kənarlıq tərzi xassəsi aşağıdakı haşiyə üslublarından birini seçməyə imkan verir:

A div element with a dashed border.

A Heading with a double border

A div element with a double border.

A Heading with a groove border

A div element with a groove border.
A Heading with a ridge border
A div element with a ridge border.
A Heading with an outset border
A div element with an outset border.
